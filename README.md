Basic Python Calculator 

Description:

This is a simple command-line calculator built using Python. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.
The program takes user input, performs the selected operation, and displays the result in a clear and user-friendly format. It also includes basic error handling, such as preventing division by zero and validating user choices.

Features:

1) Addition, Subtraction, Multiplication, Division
2) Continuous calculation using loop
3) Input validation and error handling
4) Clean and interactive user interface

Technologies Used:

1) Python

Purpose:

This project was created to practice fundamental programming concepts such as loops, conditional statements, user input handling, and basic problem-solving.
